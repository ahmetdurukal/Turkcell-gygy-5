# 📚 Library Management System

PostgreSQL ile geliştirilmiş ilişkisel kütüphane veritabanı mimarisi.

## 🎯 Proje Özeti
Sistem; öğrenci, personel, kitap, emanet ve ceza süreçlerini yönetmek için **3NF (3rd Normal Form)** kurallarına uygun tasarlanmıştır.
